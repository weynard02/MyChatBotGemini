# MyChatbot - Chat Gemini App

MyChatbot is an Android application built with Jetpack Compose that provides a conversational interface. This project aims to demonstrate a modern Android chatbot implementation, leveraging Jetpack Compose for a declarative UI and Gemini through Firebase.

## Features

*   **Chat Interface:** A clean and responsive user interface for sending and receiving messages.
*   **Message Display:** Displays chat messages from both the user and the chatbot.
*   **Markdown Support:** Renders Markdown formatted text within chat messages, allowing for rich text responses (e.g., bold, italics, lists, code blocks, links).
*   **User Input:** An input field to type messages and a send button to submit them.
*   **ViewModel Architecture:** Utilizes a `ChatViewModel` to manage UI-related data and logic, following best practices for Android architecture.
*   **Firebase Integration:** Initialized with Firebase, ready for Gemini API service.

## Technologies Used

*   **Kotlin:** The primary programming language for Android development.
*   **Jetpack Compose:** Android's modern toolkit for building native UI.
*   **AndroidX Libraries:** A comprehensive set of libraries that help develop high-quality, robust, and compatible apps.
*   **Gradle Kotlin DSL:** For build configuration.
*   **Firebase:** A platform developed by Google for creating mobile and web applications.
*   **Compose Markdown Text:** A library for rendering Markdown text in Jetpack Compose.
